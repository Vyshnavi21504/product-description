# Maska Chatpata - Spicy Peanut Butter Website

A modern, responsive, and visually appealing static website designed to showcase a premium brand of spicy peanut butter. Built with clean code practices and optimized for seamless deployment.

## Overview

**Maska Chatpata** is a professional website that promotes a premium spicy peanut butter brand. The website features a clean design, intuitive navigation, and engaging user interface elements that encourage visitor interaction through newsletter subscriptions and social media connections.

The site is built using HTML5, CSS3, and JavaScript with a focus on:

- **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- **User Engagement** - Newsletter subscription and social media integration
- **Easy Deployment** - Uses relative paths for hassle-free hosting
- **Accessibility** - Semantic HTML and SEO-friendly structure

## Tech Stack

| Technology | Purpose |

| CSS3 | Styling, responsive design, animations |
